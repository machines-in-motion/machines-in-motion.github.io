class DefaultConfiguration : public package_template::Gains_configuration
#include <default_configuration.hpp>

Default configuration for the kp, kd, ki paramters.

This class initialize the PID gains as follow:

  • kp = DEFAULT_KP,

  • kd = DEFAULT_KD

  • ki = DEFAULT_KI
