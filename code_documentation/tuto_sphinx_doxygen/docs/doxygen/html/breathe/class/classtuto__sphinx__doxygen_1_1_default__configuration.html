Class tuto_sphinx_doxygen::Default_configuration¶

class Default_configuration : public tuto_sphinx_doxygen::Gains_configuration¶

Default configuration for the kp, kd, ki paramters.

This class initialize the PID gains as follow:

  • kp = DEFAULT_KP,

  • kd = DEFAULT_KD

  • ki = DEFAULT_KI
