def cmake.CMakeTransform.parse_title (   self,
  docname 
)

Parse a document title as the first line starting in [A-Za-z0-9<] or fall back to the document basename if no such line exists.

The cmake –help-*-list commands also depend on this convention. Return the title or False if the document file does not exist.
