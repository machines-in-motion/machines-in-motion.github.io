\subsection{Detailed Description}
Convert a Y\+A\+ML file in a one line string. 

For example, a Y\+A\+ML file with the content\+: 
\begin{DoxyCode}
foo 13
bar 42
\end{DoxyCode}
 would be converted into\+:
\begin{DoxyCode}
\textcolor{stringliteral}{"\{foo: 13, bar: 42\}"} 
\end{DoxyCode}
