\subsection{Detailed Description}
Convert a Y\+A\+ML file in a one line string. 

For example, a Y\+A\+ML file with the content\+: 
\begin{DoxyCode}

2 bar 42
\end{DoxyCode}
 would be converted into\+:
\begin{DoxyCode}

\end{DoxyCode}
