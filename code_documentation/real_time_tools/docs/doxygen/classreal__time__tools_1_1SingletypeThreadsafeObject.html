Detailed Description

template<typename Type, size_t SIZE>
class real_time_tools::SingletypeThreadsafeObject< Type, SIZE >

The SingletypeThreadsafeObject is a thread safe object.

Template Parameters
Typeis the data type to store in the buffer.
SIZEis the size of the buffer. It is better to know it at compile time to be 100% real time safe.
