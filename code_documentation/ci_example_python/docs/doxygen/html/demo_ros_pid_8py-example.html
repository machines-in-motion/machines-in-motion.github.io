demo_ros_pid.py

In order to run this demos the ci_example_python/python/ci_example_python path needs to be added in the PYTHONPATH. This is done after:

  • 1. "catkin_make" is called from the root of the catkin workspace
  • 2. "source ./devel/setup.bash" is called from the root of the catkin workspace
  • 3. roscore is called in an additional terminal
  • 4. Finally run rosrun ci_example_python demo_ros_pid.py

Notice the use of the double ## for the comments. This allow Doxygen to parse you code and for you to explain in details the content of the demo.
